Tk. 2177. feladat

a. feladat

$x^{4} - 5x^{2} + 4 = 0$

  • Új ismeretlen bevezetése: $y = x^{2}$

$y^{2} - 5y + 4 = 0$

  • $a = 1, b = -5, c = 4$

$y_{1,2} = \frac{5 \pm \sqrt{(-5)^{2} - 4 \cdot 1 \cdot 4}}{2} = \frac{5 \pm 3}{2} =$

  • $y_1 = 4 \implies x^{2} = 4 \implies x_1 = 2, x_2 = -2$
  • $y_2 = 1 \implies x^{2} = 1 \implies x_3 = 1, x_4 = -1$

b. feladat

$x^{4} - 10x^{2} + 9 = 0$

  • Új ismeretlen: $y = x^{2}$

$y^{2} - 10y + 9 = 0 \implies y_{1,2} = \frac{10 \pm 8}{2}$

  • $y_1 = 9 \implies x^{2} = 9 \implies x_1 = 3, x_2 = -3$
  • $y_2 = 1 \implies x^{2} = 1 \implies x_3 = 1, x_4 = -1$

c. feladat

$x^{4} + x^{2} - 20 = 0$

$y^{2} + y - 20 = 0 \implies y_{1,2} = \frac{-1 \pm 9}{2}$

  • $y_1 = 4 \implies x^{2} = 4 \implies x_1 = 2, x_2 = -2$
  • $y_2 = -5 \implies x^{2} = -5 \implies$ Nincs valós megoldás

d. feladat

$x^{4} - 8x^{2} - 9 = 0$

$y^{2} - 8y - 9 = 0 \implies y_{1,2} = \frac{8 \pm 10}{2}$

  • $y_1 = 9 \implies x_1 = 3, x_2 = -3$
  • $y_2 = -1 \implies$ Nincs valós megoldás

e. feladat

$x^{4} - 20x^{2} - 125 = 0$

$y^{2} - 20y - 125 = 0 \implies y_{1,2} = \frac{20 \pm 30}{2}$

  • $y_1 = 25 \implies x_1 = 5, x_2 = -5$
  • $y_2 = -5 \implies$ Nincs valós megoldás

f. feladat

$x^{4} + 11x^{2} + 28 = 0$

$y^{2} + 11y + 28 = 0 \implies y_{1,2} = \frac{-11 \pm 3}{2}$

  • $y_1 = -4, y_2 = -7$
  • Mivel mindkét $y$ negatív, az eredeti egyenletnek nincs valós megoldása.

g. feladat

$64x^{4} - 20x^{2} + 1 = 0$

$64y^{2} - 20y + 1 = 0 \implies y_{1,2} = \frac{20 \pm 12}{128}$

  • $y_1 = \frac{32}{128} = \frac{1}{4} \implies x_1 = \frac{1}{2}, x_2 = -\frac{1}{2}$
  • $y_2 = \frac{8}{128} = \frac{1}{16} \implies x_3 = \frac{1}{4}, x_4 = -\frac{1}{4}$

h. feladat

$25x^{4} + 74x^{2} - 3 = 0$

$25y^{2} + 74y - 3 = 0 \implies y_{1,2} = \frac{-74 \pm 76}{50}$

  • $y_1 = \frac{2}{50} = \frac{1}{25} \implies x_1 = \frac{1}{5}, x_2 = -\frac{1}{5}$
  • $y_2 = -3 \implies$ Nincs valós megoldás

i. feladat

$x^{6} - 7x^{3} - 8 = 0$

  • Helyettesítés: $y = x^{3}$

$y^{2} - 7y - 8 = 0 \implies y_{1,2} = \frac{7 \pm 9}{2}$

  • $y_1 = 8 \implies x^{3} = 8 \implies x_1 = 2$
  • $y_2 = -1 \implies x^{3} = -1 \implies x_2 = -1$

j. feladat

$x^{6} - 28x^{3} + 27 = 0$

$y^{2} - 28y + 27 = 0 \implies y_{1,2} = \frac{28 \pm 26}{2}$

  • $y_1 = 27 \implies x^{3} = 27 \implies x_1 = 3$
  • $y_2 = 1 \implies x^{3} = 1 \implies x_2 = 1$

k. feladat

$x^{6} + 9x^{3} + 8 = 0$

$y^{2} + 9y + 8 = 0 \implies y_{1,2} = \frac{-9 \pm 7}{2}$

  • $y_2 = -8 \implies x^{3} = -8 \implies x_2 = -2$

l. feladat

$x^{6} - 4x^{3} - 5 = 0$

$y^{2} - 4y - 5 = 0 \implies y_{1,2} = \frac{4 \pm 6}{2}$

  • $y_1 = 5 \implies x^{3} = 5 \implies x_1 = \sqrt[3]{5}$
  • $y_2 = -1 \implies x^{3} = -1 \implies x_2 = -1$
